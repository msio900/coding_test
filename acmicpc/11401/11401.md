첫째 줄에 N\(N\)과 K\(K\)가 주어진다. (1 ≤ N\(N\) ≤ 4,000,000, 0 ≤ K\(K\) ≤ N\(N\))

### 출력

 (NK)\(\binom{N}{K}\)를 1,000,000,007로 나눈 나머지를 출력한다.
