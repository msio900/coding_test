### 재귀

: 하나의 함수에서 자기 자신을 다시 호출해 작업을 수행하는 알고리즘

* N부터 1까지 출력하는 함수

  ```c++
  void func1(int n){
      if (n == 0) return;
      cout << n << ' ';
      func(n-1);
  }
  ```

  

* 1부터 N까지의 합을 구하는 함수

  ```c++
  int fun2(int n){
      if (n == 0) return 0;
      return n + func2(n-1);
  }
  ```

#### 구현을 했다고 끝이 아니라 왜 이렇게 되는지 제대로 짚고 넘어가자!
